# 📥 DownloadSorter

Умный органайзер для папки загрузок, который сам наводит порядок.  
Различает фото, видео, аудио, документы и PDF, извлекает из них текст (где возможно) и раскладывает по тематическим папкам с человеческими именами.

> Больше не нужно вручную разбирать десятки файлов. Просто запустите — и наслаждайтесь порядком.

## ✨ Возможности

- 🧠 **Распознаёт содержимое**  
  Фотографии (через OCR), PDF (текст или OCR), текстовые файлы — анализируются и категоризируются.  
  Видео и аудио — извлекается дата создания (если доступна) и кладутся в отдельные папки без OCR.
- 📂 **Умная сортировка**  
  Финансы, Билеты, Работа, Фото, Видео, Аудио, Документы и другие категории — всё распределяется автоматически.
- ✏️ **Понятные имена**  
  Файлы переименовываются по шаблону `Категория_Описание_Дата.расширение`.  
  Для видео/аудио сохраняется оригинальное имя (с очисткой от мусора), если оно уже осмысленное.
- 🔍 **Локальный поиск**  
  Все обработанные файлы индексируются в SQLite. Можно искать по содержимому, названию или категории.
- 🔒 **Полная приватность**  
  Всё работает на вашем компьютере. Никакие данные не отправляются в сеть.
- ⚡ **Работает в реальном времени**  
  Следит за папкой загрузок и сразу обрабатывает новые файлы.
